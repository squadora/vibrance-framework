# Table

This component allows you to apply different styles to tables.

## Usage

Ajoutez la classe John a un élément John pour appliquer le style par défaut.

<div class="docs-demo">
    <table class="table">
        <thead>
            <tr>
                <th>Header</th>
                <th>Header</th>
                <th>Header</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Content</td>
                <td>Content</td>
                <td>Content</td>
            </tr>
            <tr>
                <td>Content</td>
                <td>Content</td>
                <td>Content</td>
            </tr>
            <tr>
                <td>Content</td>
                <td>Content</td>
                <td>Content</td>
            </tr>
        </tbody>
    </table>
</div>

## Divider

To create a separator between the table rows, add the ```.table-divider``` class.

<div class="docs-demo">
    <table class="table table-divider">
        <thead>
            <tr>
                <th>Header</th>
                <th>Header</th>
                <th>Header</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Content</td>
                <td>Content</td>
                <td>Content</td>
            </tr>
            <tr>
                <td>Content</td>
                <td>Content</td>

            </tr>
            <tr>
                <td>Content</td>
                <td>Content</td>
                <td>Content</td>
